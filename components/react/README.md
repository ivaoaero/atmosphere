# Atmosphere Component Library by IVAO

## React

### Requirements

React version: > 18.3

### How to install

```bash
npm i @ivao/atmosphere-react
```

```bash
pnpm add @ivao/atmosphere-react
```

### Configure

Import our CSS by adding this line to your index file.

```typescript
import '@ivao/atmosphere-react/dist/styles/index.css';
```
