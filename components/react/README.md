### Color convert guide

The fuselage color changed after atmodash was created.

The table provided below helps you to convert the old color to the new one.

| Old color | New color |
|-----------|-----------|
| 50        | 200       |
| 100       | 250       |
| 150       | 400       |
| 200       | 450       |
| 300       | 500       |
| 400       | 550       |
| 500       | 600       |
| 600       | 700       |
| 700       | 800       |
| 800       | 900       |
| 900       | 950       |
